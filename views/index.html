Metric-Imperial Converter

You can now convert metric to imperial and vice-versa easily.
You can either use the input box below or the address bar on top.

Example usage:

/api/convert?input=4gal
/api/convert?input=1/2km
/api/convert?input=5.4/3lbs
/api/convert?input=kg

Example return:

{"initNum":1,"initUnit":"kg","returnNum":2.2046244201837775,"returnUnit":"lbs","string":"1 kilograms converts to 2.20462 pounds"}
